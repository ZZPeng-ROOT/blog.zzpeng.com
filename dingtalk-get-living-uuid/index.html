# 开始抓包

首先打开 Fiddler

进入钉钉直播和切换清晰度时会发现产生了几个数据包

双击灰色的数据包时会展示数据包的详情

在 Request Headers 里有与如下的数据

GET /live_hp/6d946907-cd19-486f-81a4-26111dd72d1c.flv?app_type=win&auth_key=1650002637-0-0-0b9d61b65814b058fbb03580b285e1f9&cid=548a31884a42699c4756e594fdbd9ba2&live_type=0&token=27fcb5023b72b6eca57286b8cb7417f3H-szyu-OmU9aVxDWH3nn8p74ViZdHmSJjCqC4hxPLFoc7czoxsBedgIgPsOvsb6v8MPYBKVSsBMdgQjZhkksk0iirT6iiIdLT7aTZjqYkKQ=&version=6.3.35-Release.3149109 HTTP/1.1

我们需要的是下图红框中也就是加载 "/live_hp/" 和 ".flv" 中的一串数据，形似 XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX, 在下面的数据中就是 6d946907-cd19-486f-81a4-26111dd72d1c
这也就是我们所需要的 UUID 了
